# Login and Signup System

This is a simple Login and Signup mini project made using HTML, CSS and JavaScript.

## Features
- User Registration
- User Login
- Data stored in browser using localStorage
- Login authentication
- Alert messages for successful registration and login
- Simple and beginner-friendly UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. User creates an account using Signup form
2. Username and password are stored in browser localStorage
3. User can login using the same credentials
4. Success message is displayed after login

## Project Purpose
This project was created for practice and learning basic frontend development concepts like:
- Forms
- localStorage
- DOM Manipulation
- Authentication basics
