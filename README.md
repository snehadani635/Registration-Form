# User Registration-Login Form
This is a simple mini project using HTML, CSS and Javascript.
# Features
- User Registration
- User Login
- Data stored in browser using local storage
- Login authentication
- Simple and beginner-Friendly UI

# Technologies Used
- HTML
- CSS
- Javascript

# How It Works
- User registers with username and password
- Data gets stored in browser local storage
- User can login with same credentials
- Login success message is displayed
